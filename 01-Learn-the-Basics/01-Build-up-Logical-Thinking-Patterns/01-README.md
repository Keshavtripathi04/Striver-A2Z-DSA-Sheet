# Patterns

This folder contains solutions to pattern problems from the Striver A2Z DSA Sheet.

## Topics Covered

- Basic star patterns
- Number patterns
- Character patterns
- Pyramid patterns
- Inverted patterns
- Symmetric patterns

## Approach

Most pattern problems are solved using nested loops.

- The outer loop generally controls the rows.
- The inner loop controls the columns, spaces, or characters printed.

## Complexity

For most pattern problems:

- Time Complexity: O(n²)
- Space Complexity: O(1)

> Complexity may vary depending on the specific pattern.
